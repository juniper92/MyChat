# MyChat
> 파이어베이스를 활용한 채팅 어플 

</br>

##1. 제작 기간 및 참여 인원
- 2022.04 ~ 현재
- 개인 프로젝트

</br>

##2. 사용 기술
- Swift
- SwiftUI
- Firebase

- 채팅(구현중)
